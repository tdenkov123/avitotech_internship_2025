# Avitotech Backend Internship 2025

Данный репозиторий является ответом на тестовое задание для отбора на стажировку в Avito.

# Как запустить
1. При необходимости отредактируйте переменные в .env файле;
2. Поднять стек (Postgres, миграции, приложение):

```bash
$ docker compose up --build -d
```

Команда запускает сервисы `db` (PostgreSQL), `migrate` (сервис миграций) и `app` (приложение).

# Дополнительный функционал

1. Реализован эндпоинт `/healthz` для health-чека.
2. Реализован эндпоинт `POST /team/deactivate` (не описан в OpenAPI) для массовой деактивации участников команды и безопасного переназначения их открытых ревью. В ответе возвращается актуальная команда, список деактивированных `user_ids` и сводка по переназначениям.

Пример json-а запроса `POST /team/deactivate`:

```json
{
  "team_name": "test",
  "user_ids": [
    "u4",
    "u1"
  ]
}
```
